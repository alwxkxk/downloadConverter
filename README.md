# 转存代理工具
- [English document]()

## 作用及原理
- 此工具主要用于解决无法下载国外文件的问题，如Arduino IDE附加开发板安装时需要下载国外二进制包。
- 与设置代理下载不同，此工具目标是无需使用梯子，毕竟梯子对新手不友好。运行在国外服务器中下载所需文件并打包，再把压缩包放到国内网盘，使用时需下载到本地再使用。

## 使用
1. 在国外的服务器中运行脚本
2. 等待下载完毕并打包
3. 下载并运行，验证是否能在本地下载文件
